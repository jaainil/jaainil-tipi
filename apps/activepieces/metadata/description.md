## Activepieces

Activepieces is an open-source, self-hosted automation platform similar to Zapier or Make, designed for developers and teams who want full control over their workflows.

### Features
- Visual workflow builder
- Hundreds of integrations
- Webhooks, polling triggers, and scheduled flows
- Self-hosted with full data ownership
- PostgreSQL and Redis backed

### Usage
After installation, access the web interface on port **8080**, complete the onboarding flow, and start building automations.

### Notes
- PostgreSQL and Redis are deployed automatically
- Encryption and JWT secrets are generated securely by Runtipi
- Designed for production self-hosting
