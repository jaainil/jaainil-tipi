## Blinko

Blinko is a self-hosted platform for managing notes and personal knowledge with built-in authentication and database support.

### Features

- Knowledge and note management
- Authentication with NextAuth
- PostgreSQL backend
- Web-based UI
- Self-hosted and privacy-friendly

### Architecture

- Blinko web application
- PostgreSQL database
- Persistent storage for database data

### Configuration

Make sure to set:
- A valid public URL
- A secure NextAuth secret
- A strong database password
