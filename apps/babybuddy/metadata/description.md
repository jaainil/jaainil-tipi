## Baby Buddy

Baby Buddy is a self-hosted web application designed to help parents track and manage baby-related activities.

### Features

- Track feedings, sleep, diaper changes, and pumping
- Growth tracking with charts
- Multiple child profiles
- Web-based, mobile-friendly interface
- Self-hosted and privacy-friendly

### Storage

All application data, including the database and media files, is stored in a persistent volume managed by Runtipi.
