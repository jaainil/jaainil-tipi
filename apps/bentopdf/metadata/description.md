## BentoPDF

BentoPDF is a simple, lightweight service for generating PDF documents via a web interface or API.

It is designed to be easy to deploy and ideal for internal tools, automation workflows, or applications that require on-demand PDF generation.

### Features

- HTML-to-PDF rendering
- Simple HTTP API
- Lightweight container image
- No external dependencies
- Fast startup and low resource usage

### Usage

Once deployed, BentoPDF is accessible through the exposed web port and can be used directly or integrated into other applications.
