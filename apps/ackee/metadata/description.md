## Ackee

Ackee is a self-hosted, privacy-focused web analytics solution. It provides useful insights into your website traffic without tracking users across the web or storing personal data.

### Features

- Privacy-friendly and GDPR compliant
- Real-time analytics
- Self-hosted with MongoDB
- Lightweight and fast
- Simple authentication system

### Usage

After installation, open Ackee using the configured port (default: `3000`).  
Log in with the admin credentials you provided and complete the initial setup.

For production use, it is recommended to place Ackee behind a reverse proxy with HTTPS enabled.
