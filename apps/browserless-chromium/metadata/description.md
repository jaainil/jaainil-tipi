## Browserless

Browserless is a hosted headless Chromium service that allows you to run browser automation, scraping, and testing workloads over HTTP.

### Features

- Headless Chromium with DevTools protocol
- Token-based authentication
- REST and WebSocket APIs
- Compatible with Puppeteer and Playwright
- Stateless and easy to scale

### Security

A token is required to access the Browserless API.  
Configure the token during installation via the Runtipi UI.
