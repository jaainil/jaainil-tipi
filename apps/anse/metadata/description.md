## Anse

Anse is a lightweight, self-hosted AI chat web application designed to work with OpenAI and OpenAI-compatible APIs.

### Features
- Clean and minimal chat UI
- Supports custom OpenAI base URLs (OpenAI, Azure, local proxies, etc.)
- Optional site password protection
- Easy deployment with Docker
- Fully self-hosted

### Configuration
You will need an OpenAI-compatible API key.  
Optionally, you can:
- Change the model ID
- Set a custom API base URL
- Protect the UI with a password

Perfect for personal AI assistants or private deployments.
